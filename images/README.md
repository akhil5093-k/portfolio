# Project Images

This folder contains images for the portfolio projects.

## Files:
- `hangman-game.jpg` - Screenshot/image for the Hangman Game project
- `quiz-app.jpg` - Screenshot/image for the Quiz Competition Web Application

## Usage:
These images are referenced in the Projects component and will be accessible via:
- `/images/hangman-game.jpg`
- `/images/quiz-app.jpg`

## Instructions:
1. Upload your `background.jpg` file and rename it to `hangman-game.jpg`
2. Upload your `quiz.jpg` file and rename it to `quiz-app.jpg`
3. Place both files in this `public/images/` folder
